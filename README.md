#Capacity-Aware Assignment
## Prerequisites: 
* Pytorch and CUDA
* future==0.18.2
* numpy==1.18.1
* scikit-learn==0.22.1
* scipy==1.4.1
* torch==1.5.0

Use python to run train.py for experiments.

